# 混频器频谱演示

这个目录包含一个 Tkinter + Matplotlib 桌面演示程序，用于切换多种混频器配置并观察理想化频谱结果。

## 运行

```powershell
python mixer_gui.py
```

## 功能

- 四种配置：
  - 单路实数混频（和频 + 差频）
  - 正交复数混频，包含 I/Q 两路输入，可交换 LO I/Q 输出上边带或下边带
  - 单边带混频，包含一路实数输入，内部生成 Q 路 90° 移相信号，可交换 LO I/Q 输出上边带或下边带
  - 双输入独立边带混频，包含两路独立实数输入，可交换 LO I/Q 互换上下边带分配
- 输入信号支持两种输入形式：
  - 上下限
  - 中心频率 + 带宽
- 本振 LO 是点频输入，在频谱图中用黑色竖线表示。
- 输出频谱图会在本振 LO 竖线上方标出 LO 频率。
- 当 LO 频率落入不合理范围，例如与实数混频输入频带重叠导致差频穿过 0 Hz 时，状态栏会用红色文字提示警告。
- 单位支持 `Hz`、`kHz`、`MHz`、`GHz`。
- 可勾选显示负频率。
- 不显示负频率时，频率轴会围绕当前有效频谱自动缩放，不强制从 0 开始。
- 架构图附近有输入框和小频谱图，底部有更大的输出频谱图。
- 频谱用有方向的梯形显示：高边表示原输入信号频率低的部分，矮边表示原输入信号频率高的部分。
- 底部输出频谱大图会用紫红色标识频谱重叠区域，并用虚线标注输出频谱上下界。

## 说明

程序使用理想混频模型展示频率搬移关系。灰色虚线梯形表示理想 IQ 结构中被抑制的镜像边带。
